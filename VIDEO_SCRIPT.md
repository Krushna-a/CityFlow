## ğŸ¯ SECTION 2: HOW IT WORKS - HIGH LEVEL (1:30-2:30)

### Visual
- Animated flowchart:
  1. Citizen reports (pin drops on map)
  2. AI deduplicates (clusters merge)
  3. Experts discuss (chat bubbles)
  4. Recommendations generated (lightbulb icon)
  5. Action taken (checkmark)

### Script
---

**"Here's the complete workflow:**

**STEP 1: CITIZEN REPORTS**
- A citizen in Bandra sees waterlogging. Takes a photo. Uploads to the app.
- Our AI instantly analyzes: severity, water depth, exact location.
- Report is ready for submission.

**STEP 2: SMART DEDUPLICATION**
- System checks: Are there similar incidents nearby?
- If yes: Shows a beautiful prompt: 'Join existing discussion or create new?'
- User decides: Join = aggregate data. Create = new location.

**STEP 3: EXPERTS DISCUSS**
- All reports for the SAME incident appear together.
- Experts, citizens, and government officials can see:
  - All linked reports
  - Discussion thread
  - Collective insights
- Real-time collaboration.

**STEP 4: AI RECOMMENDATIONS**
- System generates actionable recommendations:
  - 'Deploy temporary pumps'
  - 'Clear nearby drains'
  - 'Temporarily block roads'
- Based on severity, water depth, and historical data.

**STEP 5: ACTION & FOLLOW-UP**
- Field workers execute recommendations.
- Mark tasks as complete.
- System tracks: What worked? How long? Cost?
- Continuous improvement loop."**
